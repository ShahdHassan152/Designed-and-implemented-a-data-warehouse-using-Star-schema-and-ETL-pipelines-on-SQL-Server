# Designed-and-implemented-a-data-warehouse-using-Star-schema-and-ETL-pipelines-on-SQL-Server

Overview


This project focuses on inventory management and purchasing analysis using Excel. The goal was to assess stock health, analyze purchasing history, and identify slow-moving or obsolete items to support operational and financial decisions.



Data Sources
El-Mehy Raw Data.xlsx

Exchange Rates.xlsx

Final Analysis & Dashboard files

PDF Business Report



Objectives
Evaluate inventory availability and stock value.

Analyze item aging based on last purchase date.

Identify inactive and slow-moving inventory.

Provide actionable insights for inventory optimization.



Key Features
Data cleaning and restructuring of raw inventory files.

Stock aging calculations (days, months, years since last purchase).

Classification of items by availability and movement.

Currency normalization using exchange rates.

Excel dashboard summarizing key inventory KPIs.

Final PDF report with insights and recommendations.



Tools & Technologies
Microsoft Excel (Advanced)

Dashboarding & Reporting



Outcome


The analysis helps management reduce dead stock, optimize purchasing decisions, and improve inventory turnover.
